LuxeSpace Interiors Website
Project Overview

LuxeSpace Interiors is a multi-page interior design website built using HTML and CSS only, showcasing professional layouts, services, and projects. The website demonstrates responsive design, Flexbox, and Grid layouts, with clean styling, borders, and user-friendly navigation.

Project Pages

Home – Hero section, About preview, Featured Projects, Services preview

About – Company story, mission, philosophy, values

Services – Grid layout of all services offered (Residential, Commercial, Space Planning, 3D Visualization, Renovation, Consultation)

Projects – Gallery showcasing completed projects using CSS Grid

Contact – Simple contact form with a success page after submission

Success Page – Displays confirmation message when a message is sent

Key Features

Multi-page structure: Home, About, Services, Projects, Contact, Success

CSS Flexbox: Used for navigation, hero alignment, centering content, and contact form

CSS Grid: Used for services and project galleries

Responsive design: Works on desktop, tablet, and mobile screens

Borders and styling: Elegant borders for cards, sections, and forms

User experience: Clear navigation, hover effects, and a success page for contact form submissions

Professional color scheme: Neutral tones (beige, cream, charcoal) for modern interior design feel

How to Use

Open index.html in a web browser

Navigate using the top menu to visit About, Services, Projects, and Contact pages

Submit a message in the contact form to see the Success page

Resize your browser to test responsiveness
